The AurixOS Boot Process
=========================

The boot up sequence of AurixOS is split up into two phases:
 - Phase 1: Initialization
 - Phase 2: Preparation

--
Phase 1 initializes basic kernel structures, all available CPU features,
and sets up interrupt handlers (though only exceptions are processed
at this point). Memory management is also initialized, and all cores
are started up and initialized if the system has multiple cores.

--
Phase 2 marks phase 1 code as free memory, loads all device drivers
and starts up credential manager together with a login prompt.
